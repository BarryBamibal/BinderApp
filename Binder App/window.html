Welcome to Binder
Binder is an app designed to make your browsing experience lightning-fast. Whenever you want to open a website, you don't want to type in the entire URL, you simply want to go to the website you have in mind. That one song you listen to a lot on youtube? You don't want to have to go to youtube and type that in, you want to type in the title and go there. Well now you can with Binder.

You simply type in a word you associate with the website you want to open, and you go there. There's also the possibility to open multiple websites at once (say they are bound to the binding "work"), and the ability to have multiple bindings open the same websites. You can even search any website you want right from this app.

Use the left boxes to choose a Binding you want linked to a certain URL. You can seperate multiple URL's with commas. Keyboard Shortcut
You can create a keyboard shortcut to Binder to launch it a lot faster, here are two methods:

Method 1 goes through chrome, so this only works while chrome is running, keep that in mind
  1. Click chrome's settings button on the top-right
  2. Go to "More Tools"
  3. Go to "Extensions"
  4. Scroll to the bottom of the page and click "Keyboard Shortcuts"
  5. Find "Binder App" in this list and click the input box next to "Launch the Binder App"
  6. Press the buttons you want to trigger Binder (check out List of keyboard shortcuts to see if it is not already taken) and make sure you select "global" in the menu on the right AFTER you entered the keyboard command

Method 2 goes through Windows, so this will always work (on Windows that is)
  1. Go to chrome://apps (simply type this into the address-bar)
  2. Right-click Binder, click "Create Shortcuts", make sure "Desktop" is checked and press "Create" (you do not have to check the taskbar checkbox as well)
  3. Go to the shortcut on your desktop and right-click it, then go to properties
  4. Click the "Shortcut" field and press the keys you want to trigger Binder (check out List of keyboard shortcuts to see if it is not already taken)
  5. Hit OK and you are done, be sure not to delete the shortcut or the keyboard shortcut will also go away

Binding Shortcuts
You can also link keyboard shortcuts to Bindings, here's how to do it:
  1. Click chrome's settings button on the top-right
  2. Go to "More Tools"
  3. Go to "Extensions"
  4. Scroll to the bottom of the page and click "Keyboard Shortcuts"
  5. Find "Launch Binding X" where X is any number
  6. Input a key binding and be sure to select "Global" in the menu on the right AFTER you inputted a key binding
  7. Go back to Binder App, hit the "Reload Bindings" button and do as the window tells you
Search Bindings

Search Bindings will allow you to, after you trigger them, search the given website.
This works by you giving a URL with %s in it, the %s is then replaced by the text you give afterwards, if you placed the %s correctly, this will then search the given website (or any other function the website may have). For example, you can search Google by giving this URL:
http://www.google.com/search?q=%s

To add your own search Bindings, you can choose one of these three methods:

Method 1:
  1. First simply visit the site and search something to store it into chrome's memory. Then right-click the omnibar and click "Manage Search Engines".
  2. Find the website you just visited in the lists (it will most likely be in the bottom list).
    If you can not find the website in the list, try method 3
  3. Click the website's URL in te right-most box and copy the URL
  4. You are done, simply paste this URL as a URL for a Binding and it will work

Method 2 (choose from a list)


Method 3:
  1. Go to the website you want to search
  2. Type %s into the search bar and hit enter
  3. Copy the URL and create a Binding with it
